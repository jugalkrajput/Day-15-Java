SQL(structured query language)
==============================
SQL is a standard language used to communicate with database servers
Database server executes commands.These commands are known as SQL queries/commands
SQL commands are divided into following four categories
1. DDL(data definition language):create,drop,alter,truncate,etc
2. DML(data manipulation language):insert,delete,update,etc
3. DQL(data query language):select
4. DCL(data control language):grant,revoke
